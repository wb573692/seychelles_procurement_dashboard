# Seychelles — Procurement Dashboard

An open-source, fully static procurement intelligence dashboard combining all four public data sources published by the [National Tender Board of Seychelles](https://www.ntb.sc). 


## Data sources

All data is sourced from the National Tender Board Seychelles web page.
