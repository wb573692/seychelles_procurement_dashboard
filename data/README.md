# data/

Pre-scraped CSV files that power the dashboard.
These **must** be committed so GitHub Actions can build `docs/index.html`.

| File | Generated by | Description |
|---|---|---|
| `ntb_tenders.csv` | `python main.py --awarded` | Awarded contracts |
| `ntb_minutes.csv` | `python main.py --minutes` | All bids per tender opening |
| `ntb_eoi.csv` | `python main.py --eoi` | EOI notices |
| `ntb_advertised.csv` | `python main.py --advertised` | Advertised tenders |

## Refresh

```bash
python main.py --all --out-dir ./data
python build.py
git add data/*.csv docs/index.html
git commit -m "Refresh data $(date +%Y-%m-%d)"
git push
```

GitHub Actions will also auto-rebuild whenever CSVs are pushed.
